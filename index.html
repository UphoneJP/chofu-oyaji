とことん楽しむ有志の会です

一緒に遊べる仲間を待っています

活動内容をご紹介します

春

卒業生を送る会　>>

卒業生と校内で遊びます。内容は毎回異なり、プロジェクターで大画面マリオカートやBBQなんてことも。

卒業記念品の贈呈　>>

卒業祝いの記念品として、毎年卒業生全員に個別名入れボールペンをプレゼントしています。

新入生歓迎会　>>

学校に不慣れな新入生の心をほぐすような歓迎会を企画しています。スモアもやりましたよー

夏

<<　スポフェス

体育館で大人と子どもで走り回ります！ドッジボール、しっぽとり、障害物競走など、親子で楽しむイベントです。

<<　夏祭り

おやじの会の一大イベントは夏祭りです。天国と地獄(ボール投げゲーム)やカレーなどの模擬店を出店し、子ども達に大人気です。

裏庭整備　>>

春や秋は気候も良いため、裏庭の手入れをすることがあります。テーブル等の整備や災害時施設としてのかまどの作り直しをしました。

親睦会　>>

かまどの定期手入れついでに火をおこし、肉やマシュマロを焼いて子ども達とも楽しんでいます。春夏秋とハイペースで活動してきた仲間をねぎらう親睦会も開催しています。

地域イベントへ出店　>>

おやじの会の資金稼ぎをしながら、近くの祭りの盛り上げ役として奮闘。今年はおんたけ祭り、すえひろ祭り、東一まつりに出店しました。

秋

<<　餅つき大会

調布大塚小学校で開催される調布大塚自治会主催の餅つき大会。おやじの会はお手伝いという形で豚汁を作っています。

<<　シーズンオフ

3か月ほどのシーズンオフ期間。来シーズンに向けての大事な充電期間です。

冬

おやじの会は常にワクワクを探しています。次は２月に催される調布大塚自治会主催の『餅つき大会』に参加予定！こども達と家庭科室で野菜を切ったり、お相撲さんと餅をついたり。きっと楽しいですよ。おや(親)じ(児童)の会は、ママもウェルカムです。仲間募集中！
